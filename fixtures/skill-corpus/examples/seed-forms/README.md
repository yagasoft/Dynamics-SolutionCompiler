# Seed Forms

`CodexMetadataSeedForms` is the neutral richer-form example for this skill.

It adds:
- a child table for related-record scenarios
- a richer work-item main form
- header controls
- multiple tabs and sections
- a quick-form reference
- a related-record subgrid

Useful example files:
- `unpacked/Entities/cdxmeta_WorkItem/FormXml/main/{c67be8a4-c475-4041-90e6-78e3ed79b018}.xml`
- `normalized/entities/cdxmeta-workitem/forms/work-item-seed-main.xml`
- `work-item-seed-main.summary.md`
- `drift.md`

Use this example when the task is about:
- FormXML interpretation
- `systemform` readback
- quick forms and subgrids
- choosing between direct Dev form mutation and tracked emitted form source
